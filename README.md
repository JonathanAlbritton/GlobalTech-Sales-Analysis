# GlobalTech Sales Analysis

This project uses Python and pandas to analyze quarterly sales data for a fictional company called **GlobalTech**. It focuses on exploring data, filtering information, and generating business insights from sales performance.

---

## Project Overview

The program works with a simulated sales dataset that includes:
- Sales transactions across multiple regions
- Product categories and individual products
- Units sold and revenue data
- Promotion information
- Some missing values to reflect real-world data issues

The analysis covers:
- Data exploration
- Filtering and grouping
- Performance comparisons
- Business insights and recommendations

---

## Key Features

### Data Exploration
- Loads and previews the dataset
- Displays structure, dimensions, and summary statistics
- Identifies missing values

---

### Data Analysis

#### Basic Metrics
- Total units sold
- Total revenue
- Average unit price

#### Filtering
- Sales from specific regions (e.g., North America)
- High-volume sales (Units > 20)
- Products on promotion
- Sales from specific time periods (February 2024)

---

### Advanced Analysis

#### Performance Insights
- Best-selling product by total revenue
- Sales performance by region
- Average units sold by category

#### Promotion Analysis
- Average sales for promoted vs non-promoted items
- Total revenue from promotions
- Comparison of promotion effectiveness

---

### Business Insights

- Top-performing product categories by region
- Average price per category
- Revenue contribution by each product (percentage of total sales)

---

### Data Quality Checks
- Identifies missing values in columns
- Calculates percentage of missing data
- Reports data quality issues

---

## Technologies Used

- **Python**
- **pandas** – data analysis and manipulation
- **numpy** – numerical operations
